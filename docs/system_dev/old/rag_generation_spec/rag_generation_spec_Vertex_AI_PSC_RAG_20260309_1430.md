# RAG生成仕様：Vertex AI PSC RAG

## 1. 入力と出力
- **入力**: ユーザーの `query`、検索結果（チャンク/抜粋）、ユーザーコンテキスト（department_id）
- **出力**:
  - `answer`（本文）
  - `citations`（根拠。最低1件。0件の場合は「見つからない」応答にする）

## 2. 生成モデル（候補）
- Gemini 1.5 Pro / Flash（要件に応じて選択）

## 3. プロンプト方針（骨子）
- 根拠（検索結果）に含まれる情報のみで回答する
- 根拠が不足する場合は「不足」を明示し、推測で断定しない
- 回答の末尾に citations を付ける（document_id/title/excerpt）

## 4. 回答フォーマット（例）
- 1段落目: 結論
- 2段落目: 根拠と補足
- 末尾: 参照（箇条書き）

## 5. ストリーミング
- `stream=false` をデフォルト
- `stream=true` の場合は Server-Sent Events（SSE）で段階配信（実装時に確定）

## 6. セーフティ・制約
- PIIの出力抑制（ルールベース/モデル設定）
- 最大トークン、タイムアウト、リトライ方針を運用設計と整合させる
