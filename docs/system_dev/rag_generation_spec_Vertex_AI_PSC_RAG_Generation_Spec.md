# RAG生成仕様：Vertex AI PSC RAG

## 用語定義
- **PSC**: Private Service Connect。

## 1. 生成方針 (Grounding Policy)
- **グラウンディング**: 必ず提供された検索結果（コンテキスト）のみに基づいて回答を生成する。
- **根拠不足時の対応**: 検索結果に回答が含まれない場合は、憶測で答えず「提供された資料の中には該当する情報が見つかりませんでした」と回答すること。

## 2. プロンプトテンプレ (Prompt Template)
```text
あなたは社内ドキュメントに基づく専門のアシスタントです。
以下の【コンテキスト】の内容のみを使用して、ユーザーの【質問】に正確に答えてください。
【コンテキスト】${contexts}
【質問】${query}
```

## 3. 出力契約 (Output Contract)
- API `/api/ask` の Response JSON 内の `answer` および `citation` 配列の形式。
- **引用形式**: `[1]`, `[2]` のように引用番号を付与。

## 4. ストリーミング仕様
- **採用有無**: **不採用** (初回リリース)。

## 5. 安全性 (Safety & Guardrails)
- **PII (個人情報) フィルタ**: Cloud DLP API を活用。
- **プロンプトインジェクション対策**: 入力クエリの長さを制限（最大 500文字）。
