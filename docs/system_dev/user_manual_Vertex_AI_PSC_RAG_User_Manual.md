# ユーザーマニュアル：Vertex AI PSC RAG システム

## 用語定義
- **PSC**: Private Service Connect。

## 1. はじめに
本システムは、社内の規定やマニュアルなどのドキュメントを対象に、AI が質問に回答する RAG システムです。すべての通信は閉域網（PSC）で行われ、セキュアな環境で利用いただけます。

## 2. 基本的な使い方
### 2.1 検索して回答を得る
1. 検索画面の入力欄に「2024年度の休暇制度について教えて」などの質問を入力します。
2. 「送信」ボタンをクリックすると、AI が社内資料から回答を生成します。
3. 回答の末尾にある [1] などの番号をクリックすると、根拠となった資料を確認できます。

### 2.2 部署限定情報の取り扱い
- あなたが所属する部署に公開されているドキュメントのみが検索対象となります。他部署の機密情報が表示されることはありません。

## 3. 管理者向け：ドキュメントの登録
1. 管理画面の「ドキュメント登録」を開きます。
2. アップロードしたいファイルを選択し、適切な「カテゴリ」および「データストア ID」を設定します。
3. 「登録」ボタンを押すと、AI が内容を解析し、数分以内に検索対象に追加されます。

## 4. トラブルシューティング
- **「情報が見つかりませんでした」と表示される**:
  - 質問内容が具体的か確認してください。
  - お使いの部署に該当する資料が登録されているか管理者に確認してください。
- **エラーが発生する**:
  - 社内ネットワーク（VPN 等）に正しく接続されているか確認してください。本システムはインターネットからはアクセスできません。
